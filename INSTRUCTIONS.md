# COMO RODAR O PROJETO

## ü™ü Windows

1. **Abra o Terminal** (PowerShell ou CMD) na pasta do projeto.
2. **Crie o ambiente virtual**:
   ```powershell
   python -m venv .venv
   ```
3. **Ative o ambiente**:
   ```powershell
   .venv\Scripts\activate
   ```
4. **Instale o Flask**:
   ```powershell
   pip install flask
   ```
5. **Rode o projeto**:
   ```powershell
   python app.py
   ```
6. **Acesse no navegador**:
   Abra o link: http://127.0.0.1:5000

## üçé macOS / üêß Linux

1. **Abra o Terminal** na pasta do projeto.
2. **Crie o ambiente virtual**:
   ```bash
   python3 -m venv .venv
   ```
3. **Ative o ambiente**:
   ```bash
   source .venv/bin/activate
   ```
4. **Instale o Flask**:
   ```bash
   pip install flask
   ```
5. **Rode o projeto**:
   ```bash
   python3 app.py
   ```
6. **Acesse no navegador**:
   Abra o link: http://127.0.0.1:5000
