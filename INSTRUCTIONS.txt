COMO RODAR O PROJETO

Siga os passos abaixo, dependendo do seu sistema operacional.

=== WINDOWS ===

1. Abra o Terminal (PowerShell ou CMD) na pasta do projeto.
2. Crie o ambiente virtual:
   python -m venv .venv

3. Ative o ambiente:
   .venv\Scripts\activate

4. Instale o Flask:
   pip install flask

5. Rode o projeto:
   python app.py

6. Acesse no navegador:
   Abra o link: http://127.0.0.1:5000


=== macOS / LINUX ===

1. Abra o Terminal na pasta do projeto.
2. Crie o ambiente virtual:
   python3 -m venv .venv

3. Ative o ambiente:
   source .venv/bin/activate

4. Instale o Flask:
   pip install flask

5. Rode o projeto:
   python3 app.py

6. Acesse no navegador:
   Abra o link: http://127.0.0.1:5000
